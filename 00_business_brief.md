references it. Swap this file out to reuse the same prompt system for a
different business.

  multilingual bridge

Tone: confident-simple, with friendly warmth (not corporate, not overly
academic — should read like a coach who believes in the student, not a
university brochure)

  generation English learners, not just a marketing line
- Free demo class before enrolling
