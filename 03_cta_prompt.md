# Prompt 03 — CTA Sections

Use after the Master System Prompt and Business Brief have been provided.

```
Using the [BUSINESS BRIEF] above, write THREE distinct CTA (call-to-action)
sections for this business's website. Each must be usable as a standalone
section/block on the page — give each one a one-line section purpose comment
before the copy.

1. BOOKING / CONTACT CTA
   - Headline (max 8 words) + 1 supporting sentence + button text.
   - Should invite a low-commitment first step (per the brief: "book a free
     demo class" — never quote a price, never promise a job outcome).

2. TRUST-BUILDING CTA
   - Headline (max 8 words) + 2 sentences.
   - Must build credibility WITHOUT fabricated stats. Use the brief's real
     differentiators (small batches, individual feedback, word-of-mouth
     reputation) as the trust signal instead of invented numbers/awards.

3. LOCATION / URGENCY CTA
   - Headline (max 8 words) + 1–2 sentences.
   - Reference the T. Nagar location and accessibility (bus/metro) as a
     genuine convenience point, not generic urgency.
   - Urgency must be honest — e.g. limited seats per batch due to the
     12-student cap — never fake countdown-timer style urgency.

For all three, write the actual button/link text (e.g. "Book My Free Demo
Class") as a separate clearly labeled line, not buried inside a paragraph.
```
