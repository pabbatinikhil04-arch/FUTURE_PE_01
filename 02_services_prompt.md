# Prompt 02 — Services Page Content

Use after the Master System Prompt and Business Brief have been provided.

```
Using the [BUSINESS BRIEF] above, write the SERVICES PAGE copy. The business
offers multiple distinct programs — treat each one as a separate service
card. For EACH service listed in the brief, produce:

1. SERVICE NAME
   - Rewrite the raw service name into a customer-facing name (not an
     internal course code). Keep it short — 3–6 words.

2. WHO IT'S FOR
   - One sentence. Name the specific person/situation (e.g. "Final-year
     engineering students about to start interviews").

3. WHAT'S INCLUDED
   - 3–4 bullet points.
   - Each bullet describes a concrete activity or outcome of the course
     (e.g. mock interviews, group discussion practice, recorded speaking
     feedback) — not vague claims like "comprehensive training."

4. WHY CHOOSE THIS PROGRAM HERE
   - 1–2 sentences connecting back to the business's differentiators (small
     batches, per-session feedback, flexible timing) — but phrase it
     specific to THIS service, not as a copy-pasted generic line.

Repeat this 4-part structure for all 5 services in the brief. Separate each
service block clearly with a heading. Keep total length per service to
roughly 80–110 words so it stays scannable on a real webpage.
```
