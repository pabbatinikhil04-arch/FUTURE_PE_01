# Prompt 04 — Tone Adaptation Layer (Reusability Prompt)

This prompt demonstrates how the SAME system + brief structure adapts to a
different tone, proving the framework is reusable across business types —
not just rewritten for one client.

```
You will rewrite the HOMEPAGE HEADLINE + SUB-HEADLINE you generated earlier,
but adapt the tone to a different business category, keeping the same
underlying value proposition logic (transformation + who it's for).

Apply these tone rules depending on category:

- FRIENDLY (salons, cafes): warmer, more casual contractions ("you're",
  "we'll"), can use light personality/humor, shorter punchier sentences.

- PROFESSIONAL (clinics, agencies): no contractions, precise vocabulary,
  no exclamation marks, outcome-focused language, slightly longer sentences
  allowed if they add clarity.

- CONFIDENT-SIMPLE (coaching institutes, trainers — this client's tone):
  short declarative sentences, speaks to the customer's specific struggle
  directly, avoids both corporate stiffness and over-casualness, sounds like
  a coach who has seen this exact problem before.

Task: Produce the same headline + sub-headline pair in all three tones so
the difference is directly visible side by side. Label each clearly as
FRIENDLY / PROFESSIONAL / CONFIDENT-SIMPLE.
```

This is the prompt that proves the system isn't a one-off script for Verve —
the same master prompt + brief structure works for a salon, a clinic, or a
coaching institute just by changing the brief and the tone field.
