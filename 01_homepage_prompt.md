# Prompt 01 — Homepage Copy

Use after the Master System Prompt and Business Brief have been provided.

```
Using the [BUSINESS BRIEF] above, write the HOMEPAGE copy for this business's
website. Produce exactly these three parts, in this order:

1. HEADLINE
   - One sentence, maximum 12 words.
   - Must state the core value proposition: what transformation the customer
     gets, not just what the business does.
   - No taglines like "Your trusted partner in..." — be concrete.

2. SUB-HEADLINE
   - One sentence, maximum 20 words.
   - Must name WHO this is for + the specific benefit they get.
   - Should make a parent, student, or job-seeker reading it think
     "this is talking to me specifically."

3. SHORT INTRO SECTION
   - 3–4 sentences, conversational, second person ("you").
   - Sentence 1: name the customer's real frustration (e.g. freezing up when
     speaking despite knowing grammar/vocabulary well).
   - Sentence 2: introduce how this business's approach is different
     (reference the small batch size / per-session feedback differentiator).
   - Sentence 3: a concrete, specific detail (batch size, location, timing
     flexibility).
   - Sentence 4: a soft transition line into the CTA (e.g. inviting them to
     see it for themselves), but do NOT write the actual CTA button text here.

Tone: confident-simple with friendly warmth, per the brief. Do not use the
words "unlock," "elevate," "journey," or "passionate."
```
